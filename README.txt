== Description ==
Clean My WP is a WordPress plugin that cleans up unnecessary data from your WordPress installation.

= Key Features =

* Disk explorer : Search and delete big files or folders in your WordPress installation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/clean-my-wordpress` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tools->Clean My WP screen to configure the plugin
